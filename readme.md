# Jar's Error Trap

Simple LUA error trap addon for WoW. 

## Installation
Same as any other WoW Addon

### Usage

Click on the minimap icon or type `/jet` to bring up the trap screen. Click on `open` to see the error details. You can copy content out of this window with good ol' ctrl-c.
